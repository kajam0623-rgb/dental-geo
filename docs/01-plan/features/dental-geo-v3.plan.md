# dental-geo-v3 Plan

## 목표
기존 GEO 분석 앱을 온다AI 수준으로 업그레이드하여, 더 풍부한 입력, 스마트한 프롬프트 선택, 시각적 결과 대시보드, 그리고 AI 기반 콘텐츠 보고서를 제공한다.

## 핵심 요구사항

### [F-01] 입력 폼 확장
- 치과 풀네임 + 단축명 두 필드
- 진료과목 최대 3개 태그 입력
- 지역명 최대 3개 태그 입력

### [F-02] 롱테일 프롬프트 선택 화면
- 입력 기반으로 20개 롱테일 프롬프트 자동 생성
- 카드 UI로 표시, 수정/삭제/추가 가능
- 10개 선택 후 스캔 진행 (온다AI 스타일)
- 프롬프트 카테고리: 지역형/증상형/비교형/추천형

### [F-03] 스캔 설정
- ChatGPT / Gemini 각각 5회 / 10회 / 20회 선택
- 총 예상 API 호출 수 미리 표시

### [F-04] 결과 시각화
- 막대그래프: ChatGPT vs Gemini SOV 비교 (Recharts)
- 꺾은선 그래프: 날짜별 점수 추이
- 검색 날짜/시간 타임스탬프
- 경쟁사 랭킹 테이블 (AI 응답 언급 병원 집계)
- 엔진 일치율 점수

### [F-05] AI 분석 보고서
- 미노출 프롬프트 원인 분석
- 콘텐츠 작성 전략 제안 (블로그/SNS/키워드)
- 수치 기반 개선 우선순위 점수
- 프롬프트 카테고리별 약점 진단

### [F-06] 히스토리 저장 (로컬)
- localStorage로 날짜별 스캔 기록 보관
- 꺾은선 그래프 자동 연동

## 기술 스택
- Next.js (기존 유지)
- TypeScript
- Tailwind CSS + Framer Motion (기존 유지)
- **Recharts** (신규 추가 - 차트)
- ChatGPT API + Gemini API (기존 유지)
